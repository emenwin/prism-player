# Sprint 计划模板

- Sprint：Sx（YYYY-MM-DD ~ YYYY-MM-DD）
- 目标（Goals）：
  - G1 …
  - G2 …
- 容量（Capacity）：团队人日 / 请假安排 / 可用时段
- 历史速度（Velocity）：近三次平均 SP = …
- 利益相关方：@

## 1. 范围（Scope）
| PBI | 标题 | 估点(SP) | Owner | 关联 PRD | 关联 TDD/ADR |
| --- | ---- | -------- | ----- | -------- | ------------- |
| PBI-1 | …… |  |  |  |  |

> 非目标（Out of Scope）：……

## 2. 里程碑与日程
- 关键里程碑：日期与交付物
- 日历：评审/发布/回顾安排

## 3. 依赖与风险
- 依赖项：系统/团队/第三方
- 风险与缓解：……

## 4. 质量与可观测性目标
- 构建/测试覆盖率目标、关键指标与告警阈值

## 5. 验收与发布
- 验收标准与数据
- 灰度、回滚方案

## 6. 完成定义（DoD）
- CI 通过（构建/测试/SwiftLint 严格模式）
- 无硬编码字符串（使用国际化）
- 文档更新（PRD/TDD/ADR/Scrum）
- 关键路径有测试覆盖与观测埋点
